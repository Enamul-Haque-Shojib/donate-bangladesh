1. What is the Document Object Model (DOM)?

The Document Object Model presents a structured, tree-like representation of a web document, where every element of the webpage, such as images, paragraphs, and headers, is represented as an object or a 'node'. This structure allows scripts, typically written in JavaScript, to interact with and manipulate the content, structure, and style of the webpage in a dynamic and interactive manner.

2. How do you select an element from the DOM?

To select an element from the Document Object Model (DOM) in JavaScript, we use methods like
Using getElementById: The getElementById method allows us to select an element by its ID. This method returns the element object if it finds an element with the specified ID, otherwise, it returns null.
Using getElementsByClassName: The getElementsByClassName method allows us to select elements by their class name. It returns a collection of all elements in the document with the specified class name.
Using getElementsByTagName: The getElementsByTagName method allows us to select elements by their tag name. It returns a collection of all elements in the document with the specified tag name.
Using querySelector: The querySelector method allows us to select the first element that matches a specified CSS selector in the document.
Using querySelectorAll: The querySelectorAll method allows us to select all elements that match a specified CSS selectors in the document.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a single event listener is attached to a parent element, it is allowing it to handle events triggered by its child elements. Instead of adding multiple event listeners to individual child elements, the event is captured at a higher level and uses event bubbling to respond to events that happen on any of its descendants.
If we have a large number of child elements, adding an event listener to each one can be resource-intensive. Event delegation allows us to manage them with just one listener.Instead of managing multiple event listeners, a single listener on the parent element can handle all events. This makes the code easier to maintain and extend.

4. How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, we first need to select the desired element using methods like getElementById, getElementsByClassName, or querySelector, then we use the setAttribute method to change attributes and the style property to modify CSS styles directly on the element.
